# Fetch Transactions and Blocks Overview

Push Chain is an EVM-compatible blockchain, allowing developers to easily interact using standard Ethereum tools. For fetching transactions and blocks, you can simply utilize the popular `viem` library, a powerful and efficient Ethereum client for JavaScript/TypeScript.

If you need more information about how to use `viem`, please refer to the official `viem` documentation.

## Fetch a Transaction by Hash

Here's how you can fetch a transaction by its hash using `viem`:

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>
```typescript
import { createPublicClient, http, defineChain } from 'viem'

// Define the Push testnet chain configuration
const pushTestnet = defineChain({
  id: 9000,
  name: 'Push Testnet',
  nativeCurrency: {
    decimals: 18,
    name: 'nPUSH',
    symbol: 'nPUSH',
  },
  rpcUrls: {
    default: {
      http: ['https://evm.pn1.dev.push.org'],
    },
  },
  blockExplorers: {
    default: {
      name: 'Push Explorer',
      url: 'https://explorer.dev.push.org/',
    },
  },
})

// Create a public client for Push testnet
const client = createPublicClient({
  chain: pushTestnet,
  transport: http('https://evm.pn1.dev.push.org'),
})

const transactionHash = '0x177482c5a504f3922875c216f71a2b236f344cfbf334f97c8f59547e1e21fb23';

const transaction = await client.getTransaction({
  hash: transactionHash,
});

console.log(transaction);
```

## Fetching Blocks

You can fetch blocks using `viem` as follows:

### Fetch Latest Block

```typescript
const latestBlock = await client.getBlock();
```

### Fetch Block by Number

```typescript
const blockNumber = 123456;
const block = await client.getBlock({ blockNumber: BigInt(blockNumber) });
console.log(block);
```

### Fetch Block by Hash

```typescript
const blockHash = '0x36939148bee59c6e1a9d4e6e6fb4e72d407f8667324714c206e64e1485f0f5ee';
const block = await client.getBlock({ blockHash });
console.log(block);
```
