PushUniversalAccountButton

Overview​

PushUniversalAccountButton is a versatile, state-aware button component for wallet connections in the Push Chain ecosystem. It handles the complete user journey from connection initiation through authentication to displaying the connected account state, with extensive customization options for each state.

Installation​

npm install @pushprotocol/pushchain-ui-kit
# or
yarn add @pushprotocol/pushchain-ui-kit

Usage​

Place the button in your UI where users should be able to connect. The button must be used within a PushWalletProvider:

import { PushUniversalAccountButton } from '@pushprotocol/pushchain-ui-kit';

function Header() {
  return (
           <Box
             display="flex"
             alignItems="center"
             justifyContent="center"
             width="-webkit-fill-available"
           >
             <PushUniversalAccountButton />
           </Box>
  );
}

Button States​

The button transitions between three main states:

  1. Disconnected: Login button prompting users to connect
  2. Loading: Connection/authentication in progress
  3. Connected: Displays the connected account with dropdown menu

Each state can be fully customized through props.

Props​

Instance Targeting​

PropertyTypeDescriptionDefault
uidstringTarget provider instance, matches config.uidUses default provider

Disconnected State Customization​

PropertyTypeDescriptionDefault
loginTextstringText for the connect buttonFrom buttonDefaults
loginBgColorstringBackground color for the connect buttonFrom buttonDefaults
loginTextColorstringText color for the connect buttonFrom buttonDefaults
loginStyleReact.CSSPropertiesInline style overrides for connect buttonFrom buttonDefaults
loginCustomReact.ReactNodeCompletely replace the default buttonNone

Loading State Customization​

PropertyTypeDescriptionDefault
loadingComponentReact.ReactNodeCustom loading indicatorFrom buttonDefaults

Connected State Customization​

PropertyTypeDescriptionDefault
walletBgColorstringBackground color for connected buttonFrom buttonDefaults
walletTextColorstringText color for connected buttonFrom buttonDefaults
walletStyleReact.CSSPropertiesInline style overrides for connected buttonFrom buttonDefaults
walletCustomReact.ReactNodeCompletely replace the connected UI-

App Preview Overrides​

PropertyTypeDescriptionDefault
modalAppOverrideObjectOverride app preview in the wrapper iframeFrom app
modalAppOverride.logoURLstringOverride app logoFrom app.logoURL
modalAppOverride.titlestringOverride app titleFrom app.title
modalAppOverride.descriptionstringOverride app descriptionFrom app.description
loginAppOverrideObjectOverride app preview inside hosted iframeFrom app
loginAppOverride.logoURLstringOverride app logo in iframeFrom app.logoURL
loginAppOverride.titlestringOverride app title in iframeFrom app.title
loginAppOverride.descriptionstringOverride app description in iframeFrom app.description

function App() {

  const walletConfig = {
    env: PushWalletConstants.ENV.TESTNET,
    login: {
      email: true,
      google: true,
      wallet: {
        enabled: true,
        chains: [PushWalletConstants.CHAIN.SOLANA, PushWalletConstants.CHAIN.ETHEREUM]
      },
      appPreview: true
    }
  };

  const appMetadata = {
    logoUrl: "https://plus.unsplash.com/premium_photo-1746731481770-08b2f71661d0?q=80&w=2671&auto=format&fit=crop&ixlib=rb-4.1.0&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D",
    title: 'Test App Title',
    description: 'Test App Description',
  };

  const modalDefaults = {
    loginLayout: PushWalletConstants.LOGIN.SPLIT,
    showModalAppPreview: true,
    bgColor: '#000'
  };

  const buttonDefaults = {
    accountMenuVariant: 'hover'
  };

  function WalletUI() {
    const { universalAddress } = usePushWalletContext();

    return (
      <div style={{ padding: '15px', borderRadius: '8px', background: '#2a2a3a', color: 'white' }}>
        <PushUniversalAccountButton />
      </div>
    );
  }

  return (
    <PushWalletProvider
      config={walletConfig}
      themeMode={PushWalletConstants.THEME.DARK}
      app={appMetadata}
      modalDefaults={modalDefaults}
      buttonDefaults={buttonDefaults}
    >
      <WalletUI />
    </PushWalletProvider>
  );
}
